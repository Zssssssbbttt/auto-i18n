# i18n 自动扫描脚本 — 使用指南

## 快速开始

```bash
# 预览模式（不修改任何文件，仅查看将要改什么）
npm run i18n:dry

# 执行扫描和替换
npm run i18n:scan
```

## 命令行参数

| 参数 | 简写 | 说明 | 示例 |
|------|------|------|------|
| `--dry-run` | `-d` | 预览模式，不写文件 | `node scripts/i18n-scan/index.cjs -d` |
| `--config` | `-c` | 指定配置文件路径 | `node scripts/i18n-scan/index.cjs -c ./my-config.js` |
| `--files` | `-f` | 只扫描指定文件（逗号分隔） | `node scripts/i18n-scan/index.cjs -f "src/views/list.vue"` |

## 配置文件 `i18n.config.js`

项目根目录下的 `i18n.config.js`，完整配置项：

```js
export default {
  // 扫描范围（glob 模式）
  entry: ['src/**/*.{vue,js,ts,jsx,tsx}'],

  // 排除文件
  exclude: ['src/router.ts'],

  // locale 文件输出目录
  output: 'src/locales',

  // 项目源码根目录
  baseDir: 'src',

  // 源码中的语言（作为 key 来源）
  sourceLanguage: 'zh-CN',

  // 要生成骨架的其他语言（添加泰语: ['en', 'th']）
  targetLanguages: ['en'],

  // 需要翻译的 HTML 属性（白名单）
  translateAttributes: [
    'label', 'placeholder', 'title', 'title-info', 'alt',
    'message', 'content', 'desc', 'text', 'header', 'menuTitle',
  ],

  // 永远不翻译的属性（黑名单）
  ignoreAttributes: [
    'style', 'class', 'ref', 'rules', 'model', 'prop',
    'key', 'slot', 'name', 'id', 'type', 'format',
    'value-format', 'range-separator', 'prefix-icon', 'suffix-icon',
    'scoped', 'lang', 'src', 'href', 'target', 'width', 'size',
    'mode', 'disabled', 'clearable', 'filterable', 'remote',
    'reserve-keyword', 'multiple', 'show-overflow-tooltip',
    'align', 'maxlength', 'rows', 'trigger', 'icon',
  ],

  // 跳过这些方法的字符串参数（不翻译）
  ignoreMethods: [
    'console.log', 'console.error', 'console.warn', 'console.info',
    'openTag', 'indexOf', 'includes', 'split', 'toString',
  ],

  // key 命名风格
  keyStyle: 'camelCase',

  // 日志输出目录
  logDir: 'logs',
}
```

## 添加新语言

1. 在 `i18n.config.js` 的 `targetLanguages` 中添加语言代码：
   ```js
   targetLanguages: ['en', 'th']  // 添加泰语
   ```

2. 运行 `npm run i18n:scan`，会自动生成 `src/locales/th.json` 骨架文件

3. 手动翻译 `th.json` 中的空字符串

```json
{
  "common": {
    "search": "查询",
    "reset": "重置",

```

## 模板替换规则

| 原始写法 | 替换后 |
|---------|--------|
| `label="申请人"` | `:label="$t('form.applicant')"` |
| `placeholder="请选择"` | `:placeholder="$t('common.pleaseSelect')"` |
| `<el-button>发起申请</el-button>` | `<el-button>{{ $t('common.createApplication') }}</el-button>` |
| `ElMessage.warning('金额不能超过原始金额')` | `ElMessage.warning($t('validation.amountExceed'))` |
| `message: '请选择申请人'` | `message: $t('validation.pleaseSelectApplicant')` |

## 注意事项

1. **先 dry-run**：执行 `npm run i18n:dry` 预览所有改动，确认无误后再 `npm run i18n:scan`
2. **$t 全局可用**：`main.ts` 中已全局注册 `$t`，模板和脚本中直接使用，无需 import
3. **语言切换**：`App.vue` 使用 vue-i18n 的 `locale` 响应式切换，无需手动刷新页面
4. **日志**：每次扫描会在 `logs/` 目录生成 JSON 日志文件，记录所有改动
